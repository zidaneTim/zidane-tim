# C1.2 - Compétences Numériques & Tableur

## Compétence visée

Utiliser les outils numériques (tableur, traitement de données, cartographie) dans un contexte professionnel.

## Contenus

### Module 1 : Fondamentaux du tableur
- Créer et organiser des données
- Utiliser les formules de base
- Mettre en forme les données
- Créer des graphiques

### Module 2 : Analyses de données
- Trier et filtrer
- Fonctions statistiques
- Tableaux croisés dynamiques
- Représentation graphique

### Module 3 : Cartographie & SIG
- Outils cartographiques
- Intégration de données géographiques
- Analyse spatiale
